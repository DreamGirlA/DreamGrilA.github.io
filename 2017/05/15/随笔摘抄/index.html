忍不住整理的语句

赵爱玲说，如果你认识从前的我，一定会原谅现在的我。一个讨厌鬼也会有他前世今生的轨迹，把这些轨迹摸清了一些，或许你内心会多一点宽容。

毛姆在《人性枷锁》中说：“人追求的当然不是财富，但必要有足以维持尊严的生活，是自己能够不受阻扰滴工作，能够慷慨，能够爽朗，能够独立。”

美国前总统奥巴马也是如此，大部分时间只穿蓝色或者灰色的西装。再一次采访中他解释了这样做的逻辑：“我努力把选择变少，我不想再吃什么或穿什么上面做决定，因为我需要做太多其他的决定。”

你们不要以为倾听别人的抱怨是件小事，肯倾听你抱怨的人，一定把你看得很重要，才会愿意接收你的负能量，安抚你的情绪，因为她希望你抱怨完之后能开心点。一个人究竟要对你怀着怎样深沉的爱意，才会愿意听你经常抱怨。

有女子说，我一生渴望被人收藏好，妥善安放，细心保存。面窝惊，免我苦，免我四下流离，免我无枝可依。

那一世，与你不期擦肩，就在你转身之间，我已远去。从此，我便欠你一次回眸。

我亦只有一个一生，不能慷慨赠予我不爱的人。之所以不想随便谈恋爱，是因为我不想在遇到对的人之前，把最好的自己给用完了。

入我相思门，知我相思苦，长相思兮长相忆，短相思兮无穷极。

这一生，只求同一处埋葬，同一处焚香。

树欲静而风不止，子欲养而亲不待。

一见倾心，半生难忘，一生执念。

若爱，请深爱，若弃，请彻底。

佛说，今生缘分，前世注定。

情不知所起，一往而情深。

实习期总结

找工作方面

  1. 兴趣 都想找一个自己感兴趣的工作，但是很难找到自己有感兴趣，有合心意的工作。

  2. 大势 所找的工作要是有发展趋势的，而不是过了几年就没用这个行业了

  3. 领导 你的直属领导和你关系最紧密，什么样的领导带你做什么事情，很重要

  4. 团队 一个好的团队是相互协作，而不是互相挖坑。

找公司方面

要去了解公司在行业里的优势是什么，发展方向，营业模式，Gartner新技术发展曲线能告诉我们新技术对人们的影响

个人发展

  1. 对于个人发展，我们很重要的是要做好每一件事情，无论大小，要认真踏实去做，可能当时不会有什么回报，并且很辛苦，但是，要相信我们做的每一件事，就像打磨一颗珍珠，每一件事情都做好，珍珠都打磨的很光亮，肯定会有一天，会有一根绳子，将珍珠串起来，这就是我们的回报。所以每一件事，我们都要好好的做。

  2. 我们不要工作时间长了之后，就懈怠了，懒散了，半年时间，如果每天只是朝九晚五机械的做事情，你的人生也没什么意义了，很快你就会发现，你和其他努力的人的不同，他们每周都有学习新的知识，都有收获，而你只是像机器一样的工作，多年后，人的发展肯定大有不同。

  3. 我们还是要多读书，什么样的人最稀缺，就是跨行的人最值钱，所以，我们不断的学习，不断的丰富自己是很有帮助的。

  4. 我们在做工作的时候，大公司可能会因为它公司规模比较大，他们的体系比较完善，人与人之间工作上的交流相互了解的不是很多，但是，对于创业型公司，分出的部门可能没有很多，体系不是那么完善，所以，我们最好能主动的了解你们工作的流程以及工作环节，知道你是在做什么，为什么做这些事情，别人在做什么，和你有什么交互，从你工作的点上，切入进去，逐渐的了解面的事情，培养自己在公司的全局观，不要只看着自己的一亩三分地，这对未来是很有帮助的。
